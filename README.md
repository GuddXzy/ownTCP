# ownTCP

跟着学 Linux 网络编程顺手写的一个 TCP 通信项目，从 socket 一路写到 epoll，
边学边写，代码可能有不少问题，欢迎指出。

## 能干啥

- 多客户端同时连接，基于 epoll 的单线程事件驱动
- 自定义二进制协议头解决 TCP 粘包
- 用户注册登录，账号信息存文件
- 客户端之间点对点发消息，服务器负责转发
- 心跳检测，客户端断了服务器能感知到
- 重复登录会把之前的踢下线，并通知对方
- 带时间戳的分级日志，调试起来方便一些

## 跑起来

需要先装 readline：
```bash
yum install readline-devel   # CentOS
apt install libreadline-dev  # Ubuntu
```

然后：
```bash
make
./server        # 终端1
./client        # 终端2，可以多开
```

## 协议结构

自己设计的一个简单协议，Header 固定8字节：

| 字段 | 大小 | 说明 |
|------|------|------|
| len  | 4字节 | Body 长度，网络字节序 |
| type | 4字节 | 消息类型，网络字节序 |

Body 根据 type 不同有不同的结构体，比如登录是 LoginMsg，聊天是 ChatMsg。

## 文件结构
```
├── server.c    # epoll 服务端，管理所有客户端连接
├── client.c    # 客户端，独立线程处理收发和心跳
├── log.h       # 几个宏封装的简单日志
└── Makefile
```
